# Job Application Tracker

A portfolio-quality frontend project for tracking job applications, built with React, TypeScript, Vite, and custom CSS.

## Features

- Add job applications with company, role, status, date applied, link, and notes
- Filter applications by status
- Search by company or role
- Sort by newest or oldest application date
- Dashboard summary cards for total and status counts
- Responsive layout for desktop, tablet, and mobile
- Local storage persistence
- Clean empty states
- Accessible form labels and keyboard-friendly controls

## Tech Stack

- React
- TypeScript
- Vite
- CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
